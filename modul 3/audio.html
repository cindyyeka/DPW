Audio HTML5

Menampilkan audio dalam HTML dapat dilakukan menggunakan tag audio. Gunakan tag source untuk menambahkan sumber media. HTML5 mendukung format audio mp3, wav dan ogg.



Penjelasan Atribut pada Tag Audio:
1. controls : Berfungsi untuk menampilkan antarmuka/tombol pemutar audio (seperti tombol play, pause, dan pengatur volume).
2. src (di dalam tag source) : Berfungsi untuk menentukan lokasi atau nama file audio yang akan diputar.
3. type : Berfungsi untuk memberi tahu browser jenis format filenya (misal: audio/mpeg untuk mp3) agar dapat diputar dengan benar.
