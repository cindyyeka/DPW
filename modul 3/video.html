Video HTML5

Menampilkan video dalam HTML dapat dilakukan menggunakan tag video. HTML5 mendukung format video mp4, webm dan ogg.



Penjelasan Atribut pada Tag Video:
1. width & height : Berfungsi untuk mengatur ukuran lebar dan tinggi area video di halaman web.
2. controls : Berfungsi untuk memunculkan tombol kontrol video (play, pause, durasi, fullscreen, volume).
3. src : Menentukan letak/lokasi dari file video yang ingin diputar.
4. type : Menentukan format video (misal: video/mp4).
