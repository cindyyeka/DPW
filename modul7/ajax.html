Uraian AJAX dan jQuery

1. Apa itu AJAX?

AJAX (Asynchronous JavaScript and XML) adalah sebuah teknik yang memungkinkan halaman web untuk mengirim dan mengambil data dari server secara asynchronous (di latar belakang). Dengan AJAX, halaman web dapat diperbarui tanpa harus memuat ulang (refresh) seluruh halaman.

2. Apa itu jQuery?

jQuery adalah library JavaScript yang sangat populer karena slogannya "Write less, do more". jQuery mempermudah penulisan kode JavaScript yang rumit menjadi lebih singkat, seperti manipulasi elemen HTML, animasi, dan mempermudah penggunaan AJAX itu sendiri.

3. Cara Menggunakan jQuery

Untuk menggunakan jQuery, kita cukup memasukkan link library-nya di bagian <head> atau sebelum penutup </body> menggunakan CDN (Content Delivery Network).
